# Lucky Kumar - VR Developer & Gaming Technology Student Portfolio

Welcome to the repository of my personal portfolio website! This project showcases my skills, experience, projects, and achievements as a VR/AR Developer and Gaming Technology student. 

The website highlights my background in developing immersive virtual reality experiences, games, and web-based applications, as well as providing a way to get in touch.

## 🚀 Live Demo

https://portfolio-website-eight-bay-36.vercel.app/

## 🛠️ Built With

This project relies on a modern frontend tech stack:
* **Framework:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Routing:** [React Router](https://reactrouter.com/)
* **Contact Form Integration:** [Web3Forms](https://web3forms.com/)
* **Icons:** [Remix Icon](https://remixicon.com/)

## 🌟 Key Features

The portfolio is divided into several informative, beautifully styled sections:

- **Hero Section:** A captivating introduction to my identity as an Immersive Experience Creator.
- **Profile Overview:** Details my academic qualifications, current status, and key personal highlights.
- **Skills Section:** Visual breakdown of technical skills, categorized into tools/engines (e.g., Unity, AR Foundation) and languages (e.g., C#, Python).
- **Projects Showcase:** An interactive gallery of my VR, AR, Game Development, and Web projects, complete with categorized filtered views. Highlights include "Echoes in the Void", "The Woods VR", and AR simulations.
- **Experience/Achievements:** Key acknowledgments, such as my time as an XR Research Intern at IIT Delhi.
- **Contact section:** A fully functional email integration utilizing Web3Forms complete with my social media links (LinkedIn, GitHub, Instagram, Facebook).

## 📂 Project Structure

* `/public` - Houses static assets like project images and the profile picture.
* `/src/pages` - Page components (e.g., Home page, Not Found page).
* `/src/pages/home/components` - Isolated UI components that make up the single-page experience (HeroSection, ProjectsSection, ContactSection, etc.).
* `/src/router` - Application route configuration using React Router.
* `/src/index.css` - Global CSS configurations.
